Divunity : Bâtiments

Created by Clément Topy, last modified on mars 30, 2021

Il y a 3 types de ressources :

  • Bois (arbres)

  • Pierre (rochers)

  • Minerai (rochers avec minerai)

Tous les joueurs peuvent avoir du bois, mais chaque joueur n’a accès qu'à un type de rocher (pierre ou minerai).

Les rochers aparaissent comme des rochers classiques sur la map mais se transforment en minerai lorsqu’ils entrent dans la zone d’influence du joueur minerai.

Si il y a marqué “Spé”, c’est que le joueur doit utiliser sa ressource spéciale pour crafter un bâtiment (rocher ou minerai)

Si il y a marqué “Autre Spé” dans le craft d’un bâtiment, c’est qu’il faut la ressource spéciale de l’autre joueur. Le seul moyen qu’ont les joueurs de s'échanger des ressources est de les mettre dans l’entonnoir de l'étagère.
